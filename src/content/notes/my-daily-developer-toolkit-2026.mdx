Your development setup is more than just a set of utilities; it is the environment in which you spend hours every day. Over the years, I have optimized my setup for ergonomics, speed, and cognitive load. Here is the actual setup I use daily in 2026.

## Hardware Foundations

Before we talk about software, let's address the physical space. The tools you touch have the most immediate effect on your comfort and long-term health.

- **Keyboard**: A custom mechanical split keyboard (Voyager by ZSA) with brown tactile switches. It keeps my wrists straight and helps prevent RSI.
- **Mouse**: Logitech MX Master 3S. The scroll wheel makes traversing long source files a breeze.
- **Monitor**: Dell UltraSharp 40-inch Curved WUHD. Having one large screen instead of dual monitors lets me arrange windows naturally with snapping tools.

## Core Software & Editors

Moving onto the screen, these are the programs that remain open constantly:

1. **Editor**: I split my time between VS Code (configured to be ultra-minimal, with status bars hidden) and the new Zed editor for fast typing and quick file changes.
2. **Terminal**: Warp Terminal. It feels like a modern text editor rather than a retro console, allowing block-by-block execution and inline AI command generation.
3. **Shell**: Fish shell paired with Starship prompt. The auto-suggestions and clean formatting make command line navigation painless.

## Daily Utilities

Lastly, a few small tools that save minutes every hour:

- **Raycast**: The replacement for macOS Spotlight. It handles window management, clipboard history, quick calculations, and script commands.
- **TablePlus**: A clean, native database client that connects to PostgreSQL and SQLite instances without cluttering the screen.
- **Arc Browser**: Keeps tab clutter hidden and organizes different spaces for personal projects, client work, and research.
